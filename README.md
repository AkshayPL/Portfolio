---

## 🚀 Key Highlights & Features

- **Interactive 3D Wave Space background**: Dynamic, hardware-accelerated particle system built with **Three.js** that scales gracefully across mobile and desktop browsers.
- **Responsive Systems Grid**: Interactive cards highlighting custom C# mechanics, physical hardware integrations, and networking solutions.
- **WebXR Immersive VR Mode**: Built-in VR fly-through simulation compatible with WebXR-enabled headsets and browsers. When entered, all non-immersive UI is automatically hidden.
- **Interactive CV Timeline**: Modern timeline-based resume detailing professional milestones and engineering expertise.
- **Vanilla Performance**: Built using zero bulky frameworks—leveraging raw HTML5, custom CSS variables, and native ES6 Javascript to achieve ultra-fast page loads.
